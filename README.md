# learn_leetcode

My LeetCode solutions and notes.

## Goals
- Practice data structures & algorithms
- Track progress over time
- Keep clean, explainable solutions

## Suggested structure
You can organize by language, topic, or problem number. Example:

```text
learn_leetcode/
  README.md
  solutions/
    0001-two-sum/
      solution.py
      notes.md
```

## Conventions (recommended)
- One folder per problem: `####-problem-slug/`
- Include a short `notes.md` for approach, complexity, and edge cases
- Name files `solution.<ext>` (or `solution1.<ext>`, `solution2.<ext>` for variants)

## How to use
- Add your solutions under `solutions/`
- Commit as you go
